# Android 应用图标更新指南

## 方案 1：使用在线工具（推荐）

访问 [Android Asset Studio](https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html)

1. 上传 `assets/temp_icon.png`
2. 调整 padding 和样式
3. 下载生成的 zip 包
4. 解压到 `android/app/src/main/res/`（覆盖现有文件）

---

## 方案 2：使用脚本（需要 ImageMagick）

```bash
cd frontend/web-app

# 安装 ImageMagick（如果没有）
# Ubuntu/Debian: sudo apt install imagemagick
# macOS: brew install imagemagick
# Windows: choco install imagemagick

# 生成图标
chmod +x scripts/generate-android-icons.sh
./scripts/generate-android-icons.sh assets/temp_icon.png
```

---

## 方案 3：手动替换（最简单）

需要的尺寸：

| 密度 | 尺寸 | 目录 |
|------|------|------|
| mdpi | 48x48 | `mipmap-mdpi/` |
| hdpi | 72x72 | `mipmap-hdpi/` |
| xhdpi | 96x96 | `mipmap-xhdpi/` |
| xxhdpi | 144x144 | `mipmap-xxhdpi/` |
| xxxhdpi | 192x192 | `mipmap-xxxhdpi/` |

每个目录需要 3 个文件：
- `ic_launcher.png` - 标准图标
- `ic_launcher_round.png` - 圆形图标
- `ic_launcher_foreground.png` - 前景图标（用于 adaptive icon）

使用图片编辑工具（Photoshop/GIMP/在线工具）调整 `temp_icon.png` 到对应尺寸，保存到对应目录。

---

## 验证

```bash
cd android
./gradlew assembleDebug
# 安装 APK 后检查应用图标
```
